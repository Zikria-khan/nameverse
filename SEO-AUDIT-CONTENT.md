## 10.2 High-Impressions Low-CTR Pages

### Name Pages (200K impressions, 5% CTR)

**Current Title Pattern:**
```
{Name} — {Modifier} Meaning | NameVerse
```
Examples:
- `Muhammad — Name Meaning | NameVerse`
- `Aisha — Arabic Origin | NameVerse`
- `Ali — Quranic Name | NameVerse`

**Issues:**
1. Titles are generic and template-like
2. Missing year/trend signals ("2026")
3. Missing specific meaning keywords
4. Brand name "NameVerse" takes 10 characters
5. No emotional triggers or power words

**Recommended Title Templates:**
```
Template 1: {Name} Name Meaning — {Origin} {Religion} Baby Names 2026
Template 2: {Name} — {Meaning} | {Religion} Baby Name Guide
Template 3: {Name} Baby Name: Meaning, Origin & Lucky Number | NameVerse
Template 4: {Name} — {Religion} Name Meaning & Cultural Significance
```

**Current Meta Description Pattern:**
```
{Name} is a {gender} name from {religion} tradition meaning "{meaning}". Explore detailed meaning, origin and cultural history on NameVerse.
```

**Issues:**
1. Generic template structure
2. Missing specific call-to-action
3. No trust signals (verification stats)
4. Missing year references

**Recommended Meta Templates:**
```
Template 1: Discover {Name} meaning: {Religion} {gender} name meaning "{meaning}" — {origin} origin. 98% verified meanings, lucky numbers, pronunciation & similar names. Free guide.
Template 2: {Name} name meaning: {Religion} {origin} baby name with meaning "{meaning}". Explore cultural significance, numerology, and 50+ similar names.
Template 3: {Name} — {Religion} name meaning "{meaning}". {origin} origin, lucky number, pronunciation, and related names. Trusted by 10,000+ parents.
```

### Religion Pages (20K impressions, 10% CTR)

**Current Title:** `{Religion} Baby Names - Boy & Girl Names with Meanings | NameVerse`
**Issue:** Generic, no year signal, no differentiation

**Recommended:** `{Religion} Baby Names 2026 — {Count}+ Verified Names with Meanings & Lucky Numbers`

### Hub Pages (15K impressions, 4% CTR)

**Current Issues:**
- `/trending-names`: Title is good but missing year specificity
- `/unique-names`: Generic title
- `/languages`: Missing specific language keywords in title
- `/popularity`: Missing data/year signals

# SECTION 11: NAMEVERSE-SPECIFIC SEO ANALYSIS

## 11.1 Name Pages as Semantic Entity Database

### Current Entity Attributes

Each name page includes:
- `name` (string)
- `short_meaning` / `meaning` (string)
- `origin` (string)
- `religion` (string: islamic/christian/hindu)
- `gender` (string: male/female/other)
- `pronunciation.english` (string, optional)
- `pronunciation.ipa` (string, optional)
- `lucky_number` (number)
- `languages` (array, optional)
- `name_variations` (array, optional)
- `similar_sounding_names` (array, optional)
- `published_date` / `created_at` / `updated_at` (date)

### Missing Entity Attributes

1. **Popularity score** — No historical or current popularity data
2. **Famous people** — No celebrity or historical figure associations
3. **Name day** — No cultural name day celebration date
4. **Numerology details** — Only lucky number, no full numerology reading
5. **Astrological association** — No zodiac or planetary associations
6. **Syllable count** — Missing phonetic structure data
7. **Name length categories** — Not tagged (3-letter, 4-letter, etc.)
8. **Variant spellings** — `name_variations` exists but not linked
9. **Cultural taboos** — No information on names to avoid in certain cultures
10. **Name combinations** — No sibling name pair suggestions
11. **Surname compatibility** — No "sounds good with" analysis
12. **Middle name suggestions** — No data
13. **International pronunciation** — Only English + IPA
14. **Written forms** — No Arabic script, Devanagari, etc.
15. **Etymology tree** — No root word decomposition
16. **Related concepts** — No thematic associations (e.g., "names meaning light")
